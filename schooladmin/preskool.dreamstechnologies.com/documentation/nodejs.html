Installation Guide

Node.js and NPM:
  • If NodeJS is not installed then download it from below link and NPM comes bundled with Node.js

  • https://nodejs.org/

NPM Packages:

Before proceding you'll need to install npm packages. You can do this by running npm install from the root of your project to install all the necessary dependencies. 

 npm install
Development Server:

Run npm run start for a dev server. Navigate to localhost:3002/. The app will autometically reload if you change any of the source files. 

npm run start
Build Project:
  • Run npm run build to build the project.
  • After that you will get dist folder inside source code folder
  • Copy everything within the output folder (dist/ by default) to a folder on the server.
  • Now you can run this Template in any web.
